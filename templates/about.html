About This Project

What kind of film? is a smart movie search and recommendation tool that helps you explore different types of films based on your own descriptions.

Whether you type in actor names, movie titles, genres, or just describe a scene â€” our system uses Natural Language Processing (NLP) and Latent Semantic Analysis (LSA) techniques to understand your intent and suggest related films.

Technologies Used

  • HTML, CSS, and JavaScript for frontend
  • Flask for backend
  • Latent Semantic Analysis (TF-IDF, BoW, Word2Vec, etc.)
  • Voice input via Web Speech API

Purpose

This project was developed for educational purposes to explore modern techniques in information retrieval, vector embeddings, and user interaction through voice and text.
